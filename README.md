# bissli
Bissli is  a snack based project to easily create TUI menus

# Prerequisites 
Installed python snack package 

$pip install snack 

# Example 
The code will only run with python2.X 
```
$ cd examples
#make sure that the bissli directory in in your python path.```
$ python
$ import fastfood 
```
